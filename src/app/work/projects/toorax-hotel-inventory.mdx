## Problem

A full-service hotel needed **inventory truth** tied to kitchens, bars, and stores — with staff working on-site and low tolerance for cloud latency or internet outages during operations.

## Tech stack

- **Django** backend services
- **Next.js** responsive frontend
- **On-premises** deployment with secure network boundaries
- Integration touchpoints for **local hardware** and internal operational tools

## What I built

- End-to-end technical execution from **system analysis** through **production**.
- **Microservice-oriented** decomposition so stock, purchasing, and reporting could evolve separately.
- **DevOps** for on-site servers: backups, updates, and monitoring appropriate for a single-property operator.

## Outcome

**Low-latency, reliable inventory** aligned with how the hotel actually runs — not a cloud demo that fails when the uplink blinks.
